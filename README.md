# Кружки — мультиплеер

Простая мультиплеерная игра: каждый подключившийся клиент получает
кружок-игрока, который может ходить по клеточной карте (20×15).
Синхронизация состояния — через WebSocket в реальном времени.

## Запуск

1. Установить Node.js (если ещё нет): https://nodejs.org
2. В папке проекта выполнить:

   ```bash
   npm install
   npm start
   ```

3. Открыть в браузере: http://localhost:8080
4. Открыть эту же страницу в нескольких вкладках/устройствах в той же сети —
   каждая вкладка получит свой кружок.

## Управление

Стрелки ← ↑ → ↓ или W A S D.

## Как устроено

- `server.js` — Node.js + `ws`. Держит состояние всех игроков (координаты,
  цвет), обрабатывает сообщения о движении, проверяет границы карты и
  занятость клеток, рассылает всем клиентам актуальное состояние.
- `index.html` — рисует сетку и кружки на `<canvas>`, отправляет на сервер
  события нажатий клавиш, перерисовывает сцену при каждом обновлении
  состояния от сервера.

## Что можно добавить дальше

- Препятствия/стены на карте
- Чат между игроками
- Имена игроков (ввод при подключении)
- Разные карты/комнаты
- Деплой на Render/Railway/Fly.io, чтобы играть не только в локальной сети
